## CH9329设备

CH9329 芯片是由沁恒生产的一种串口转HID键盘鼠标芯片。它可以将上位机发送的串口数据转换为标准的USB键鼠设备信号，并将其发送给下位机，从而实现硬件级别的键鼠模拟。
